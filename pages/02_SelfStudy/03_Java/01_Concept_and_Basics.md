# 01. Javaの概念と基礎（Web技術との対比）

このドキュメントでは、Webアプリケーション（Next.js/React等のモダン技術）のコンテキストから見た、Java言語の立ち位置や特性について整理します。

## 1. Javaとは何か？

Javaはオブジェクト指向プログラミング言語の代表格であり、「一度書けばどこでも動く (Write once, run anywhere)」を理念としています。

- **静的型付けの厳密さ**: TypeScript以上に厳格な型システムを持ち、コンパイル時にエラーを検知することに重きを置いています。
- **JVM上で動作**: 実行環境であるJava Virtual Machine (JVM) を経由して動くため、OSに依存しない堅牢なバックエンドシステムの構築で今でも主流です。

## 2. Web技術との対比

- **Spring BootとNext.js**: Javaの世界では「Spring Boot」という巨大なフルスタックフレームワークがよく利用されます。Next.jsがフロントエンド主導であるのに対し、Spring Bootは強力なバックエンドAPIと高度なDI（依存性の注入）を中心に据えています。
- **エコシステムの成熟度**: 銀行やエンタープライズ領域などの大規模システムでの実績が非常に多く、古いコード資産から最新のモダンアーキテクチャまで幅広く存在します。
