# 01. Visual Basic (VB)の概念と基礎

Visual Basic（VB）は、Microsoftが開発したイベント駆動型プログラミング言語および環境です。Excel VBAと根源を同じくしています。

## 1. VBとVBAの違い

- **Visual Basic (VB / VB.NET)**: 独立したWindowsアプリケーション全体を開発するための強力な言語です。「.NET」フレームワーク上で動くモダンな用途（VB.NET）として進化しました。
- **VBA (Visual Basic for Applications)**: ExcelやAccessなど、Office製品の「中」に寄生して動くマクロ言語です。言語の文法はVBとほぼ共通ですが、実行環境が制限されています。

## 2. Web開発者の視点から見た特徴

- **GUIの作りやすさ**: Visual StudioというIDEを使うことで、ボタンやテキストボックスをドラッグ＆ドロップで配置し、ダブルクリックして処理（イベントハンドラ）を書くという、Reactのコンポーネント指向に近いような「直感的なUI開発」ができるのが最大の特徴です。
- **レガシーシステムとの関わり**: かつて企業内で大量に開発されたWindows向け業務ツールの多くがVB（VB6等）で書かれており、その保守やWebシステムへの移行という文脈で出会うことが多い言語です。
