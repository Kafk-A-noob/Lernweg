# Lernweg (学習の軌跡)

プログラミング未経験からの学習の軌跡、および実務を見据えた技術ナレッジを集約したパーソナル・ドキュメントサイトです。
日々の学習ログから、実務で使えるお作法・メソッド図鑑まで、フルスタックエンジニアとしての道を記録しています。

---

## カテゴリガイド

Lernwegは大きく分けて以下の領域で構成されています。

### 1. 実践プロジェクト (Projects)

- **Web Application**: AI完全生成によるWebアプリ・ツールの開発ログ
- **Portfolio**: メンターとの協業によるWeb開発ログ (Next.js / 3Dモデル統合等)
- **Infrastructure**: 自作物理サーバー＆Docker等のインフラ構築ログ

### 2. カリキュラム・学習ログ (25R1116)

職業訓練校および自己学習における、日々の講義内容や課題の記録です。

- **Frontend**: HTML/CSS, React, CSSフレームワーク
- **Backend**: Flask (Python), Express (Node.js)
- **[Infrastructure / Linux](./01_25R1116/03_Infrastructure/01_Linux_Ubuntu/01_Network_and_Hosts)**: Ubuntu, WSL, Apacheと権限管理の基礎
- **[Coding Test](./01_25R1116/04_Others/02_CodingTest/01_標準入力とCLI実行の基礎知識)**: CLIアプリケーションにおける各言語の標準入力パースと自動採点対策

### 3. 実務・お作法図鑑 (Encyclopedia)

**【New!】** 各言語特有のお作法や罠、実用的なメソッドを体系化した辞典のようなものです。

- **[JavaScript / TypeScript](./03_Encyclopedia/01_JavaScript_TypeScript/01_JS_TS_ロジック構築メソッド図鑑)**: 非同期処理の思想、型システム
- **[Python](./03_Encyclopedia/02_Python/01_Python_ロジック構築メソッド図鑑)**: リスト内包表記、Pythonicな構文
- **[PHP](./03_Encyclopedia/03_PHP/01_PHP_ロジック構築メソッド図鑑)**: 配列と連想配列の同一視、テンプレート挙動
- **[Frameworks](./03_Encyclopedia/04_Frameworks/01_React_Nextjs_逆引き辞典)**: React/Next.js, Flask, Laravel
- **[Mobile App](./03_Encyclopedia/06_Mobile_App/01_Native/01_Swift_Kotlin_概要と歴史)**: Native (Swift/Kotlin), CrossPlatform (Flutter)
- **[Database](./03_Encyclopedia/05_Database/01_SQL_RDB_基本クエリとアンチパターン図鑑)**: N+1問題、トランザクション、論理削除設計

### 4. 自己学習 (Self-Study)

訓練校のカリキュラム外での、VBAやAccess等の自己学習記録です。

- **[Excel VBA](./02_SelfStudy/01_ExcelVBA/01_Concept_and_Basics)**: Webフロントエンド技術（React等）とのパラダイムの違い
- **[Access](./02_SelfStudy/02_Access/01_Concept_and_Basics)**: フルスタックアプリ（バックエンド・フロントエンド統合）としての対比
- **[Java](./02_SelfStudy/03_Java/01_Concept_and_Basics)**: バックエンドの巨匠言語、JVMアーキテクチャや厳格な型システム
- **[Visual Basic](./02_SelfStudy/04_VB/01_Concept_and_Basics)**: GUI開発の直感性と、VBAのネイティブ言語としての全体像
