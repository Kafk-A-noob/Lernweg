# Lernweg (学習の軌跡)

プログラミング未経験からの学習の軌跡、および実務を見据えた技術ナレッジを集約したパーソナル・ドキュメントサイトです。
日々の学習ログから、実務で使える「お作法・メソッド図鑑」まで、フルスタックエンジニアとしての道を記録しています。

---

## 🧭 カテゴリガイド

Lernwegは大きく分けて以下の領域で構成されています。

### 1. 📖 実務・お作法大図鑑 (Encyclopedia)

**【New!】** 各言語特有の「お作法」や「罠」、実用的なメソッドを他言語と比較しながら体系化した最強の虎の巻です。

- **[JavaScript / TypeScript](./01_25R1116/05_Encyclopedia/01_JavaScript_TypeScript)**: 非同期処理の思想、型システム
- **[Python](./01_25R1116/05_Encyclopedia/02_Python)**: リスト内包表記、Pythonicな構文
- **[PHP](./01_25R1116/05_Encyclopedia/03_PHP)**: 配列と連想配列の同一視、テンプレート挙動
- **[Frameworks](./01_25R1116/05_Encyclopedia/04_Frameworks)**: React/Next.js, Flask, Laravel
- **[Mobile App](./01_25R1116/05_Encyclopedia/06_Mobile_App)**: Native (Swift/Kotlin), CrossPlatform (Flutter)
- **[Database](./01_25R1116/05_Encyclopedia/05_Database)**: N+1問題、トランザクション、論理削除設計

### 2. 💻 カリキュラム・学習ログ (25R1116)

職業訓練校および自己学習における、日々の講義内容や課題の記録です。

- **Frontend**: HTML/CSS, React, CSSフレームワーク
- **Backend**: Flask (Python), Express (Node.js)
- **Coding Test**: CLIアプリケーションにおける標準入出力対策

### 3. 🚀 実践プロジェクト (Projects)

- **Web Application**: AI完全生成によるWebアプリ・ツールの開発ログ
- **Portfolio**: メンターとの協業による本格的Web開発ログ (Next.js / 3Dモデル統合等)
- **Infrastructure**: 自作物理サーバー＆Docker等のインフラ構築ログ

---

> Lernweg はドイツ語で「学びの道」を意味します。
> このドキュメントは、過去の自分が躓いたエラーを未来の自分が爆速で解決するための「道しるべ」として更新され続けます。
